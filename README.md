Sistema de Reconocimiento Óptico de Caracteres (OCR)
1. Descripción General
Este proyecto consiste en el desarrollo de un pipeline de OCR (Optical Character Recognition) diseñado para la extracción de texto desde imágenes de páginas de libros, documentos técnicos y textos académicos. El sistema utiliza modelos de aprendizaje profundo a través de la librería EasyOCR, complementado con técnicas de preprocesamiento de imágenes mediante OpenCV para optimizar la precisión de los resultados.

2. Requisitos del Sistema
Python 3.8 o superior.

Acceso a internet para la descarga inicial de los pesos del modelo pre-entrenado.

Sistema operativo: Windows, macOS o Linux.
